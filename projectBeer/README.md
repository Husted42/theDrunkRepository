## Deployment
To deploy this project you need to have the modules
- flask 
- psycopg2

```bash
$ pip install flask 
$ pip install psycopg2
```
Run the files in sqlQueries in your database (pgadmin): 
- insertData.sql

After installing the modules and having all the files, run app.py: 
```bash
$ python3 app.py
```
